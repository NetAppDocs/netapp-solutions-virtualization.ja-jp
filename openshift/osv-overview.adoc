= Red Hat OpenShift Virtualizationについて
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Red Hat®OpenShift® Virtualizationにより、組織は従来の仮想マシン（VM）を最新のワークロードと並行して、統合されたハイブリッドクラウドアプリケーションプラットフォーム上でシームレスに移行、実行、管理できるようになります。OpenShift Virtualizationは、あらゆるOpenShiftサブスクリプションに含まれるオペレーターとして提供されています。OpenShift Container Platformへのこのアドオンは、VMとコンテナ化されたアプリケーションを統合するための堅牢なソリューションを提供します。信頼性の高いLinuxカーネルハイパーバイザーであるKernel-based Virtual Machine（KVM）を使用しており、堅牢なパフォーマンス、拡張性、安定性で知られ、OpenShift内のネイティブオブジェクトとして仮想マシンのエンタープライズクラスの仮想化を実現します。

image:redhat-openshift-044.png["OpenShift仮想化"]



[[key-features-of-red-hat-openshift-virtualization]]
== Red Hat OpenShift Virtualization の主な機能

**ハイブリッドワークロード：** クラウドネイティブのメリットを活かして仮想マシンを移行および実行し、統一されたツールセットを使用してコンテナやサーバーレスワークロードと並行して管理します。

**段階的な最新化：** 既存の仮想マシンに影響を与えることなく、コンテナの採用、マイクロサービスの導入、AIの統合などによって、自分のペースで最新化を進めることができます。

**Kubernetes統合：** Kubernetesカスタムリソースを活用して、OpenShiftクラスタに仮想化機能を追加します。



[[capabilities-of-red-hat-openshift-virtualization]]
== Red Hat OpenShift Virtualization の機能

**VM管理：** LinuxおよびWindowsのVMを作成および管理します。

**統合ワークロード：** クラスター内でポッドとVMのワークロードを同時に実行します。

**接続性：** さまざまなコンソールおよびCLIツールを使用してVMにアクセスできます。

**VM操作：** ノード間でVMをインポート、クローン、ライブマイグレーションします。

**リソース管理：** VMに接続されているネットワークインターフェイスとストレージディスクを管理します。



[[compatibility]]
== 互換性

OpenShift Virtualizationは概念的にはVMware vSphereと異なりますが、仮想化管理者に特化して調整された最新のプラットフォームを提供し、以下のような使い慣れた機能を備えています。**ライブマイグレーション：** vSphereのvMotionに相当します。

**ストレージ管理：** OpenShift のストレージクラスは、vSphere のストレージポリシーベース管理（SPBM）に対応しています。

**永続ストレージ：** 永続ボリューム要求 (PVC) と永続ボリューム (PV) は、vSphere のデータストアに対応します。
