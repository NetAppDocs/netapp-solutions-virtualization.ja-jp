= NetApp Tridentと Red Hat OpenShift の統合について学ぶ
:hardbreaks:
:allow-uri-read: 
:nofooter:

[role="lead"]
OpenShift Virtualizationソリューション向けにアプリケーションおよび永続ストレージ管理のために検証されたNetApp Trident Protectについて学びましょう。

Trident、NetAppが管理するオープンソースのストレージプロビジョナーおよびオーケストレーターとNetApp Trident Protectは、Red Hat OpenShiftなどのコンテナベースの環境で永続的なデータのオーケストレーションと管理を支援します。
